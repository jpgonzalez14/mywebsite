Software Architecture

Class project for oil company that wants to acquire a software for the processing and automated monitoring of data on its wells, fields and regions of oil exploitation.

  • Implement load balances with nginx to increase the amount of request per second from 3000 to almost 7500 requests.

  • Implementation of symmetric encryption for sensitive data provided by a micro-controller.

  • Different load tests performed with apache jmeter to observe the amount of successful request per second and its response time.
