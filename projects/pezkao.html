ABOUT

Marine Life Conservation App. This application showed different species of fish in their respective state (invasive, endangered, non-endangered) and depending on the state provided the user with useful information. For example, if it was in danger, it gave information on how to help conserve the species and showed consumption alternatives with recipes.

  • Full stack application, mongo data base, node + react back end, ionic framework as front end that consume rest services.

  • Front end iteration due to user experience feedback to create a easy to use app.
